
**需要添加的JS逻辑**：
```javascript
data: {
  inviteCode: '',
  parentScoutInfo: null
},

// 验证邀请码
async verifyInviteCode() {
  const res = await wx.cloud.callFunction({
    name: 'scout',
    data: {
      action: 'verifyInviteCode',
      data: { inviteCode: this.data.inviteCode }
    }
  });

  if (res.result.valid) {
    this.setData({ parentScoutInfo: res.result.scout });
  }
},

// 注册时传递邀请码
submitRegister() {
  // ...
  const data = {
    ...this.data.formData,
    inviteCode: this.data.inviteCode  // 🆕 传递邀请码
  };
}
```

## 💡 关键设计点

### 1. 层级限制
- ✅ 只支持两级：一级星探和二级星探
- ✅ 二级星探不能再邀请下级
- ✅ 注册时自动检查上级层级

### 2. 两种码的区别
- **推荐码（shareCode）**：所有星探都有，用于推荐候选人
- **邀请码（inviteCode）**：只有一级星探有，用于邀请二级星探

### 3. 数据一致性
- ✅ 注册二级星探时，自动更新一级星探的团队统计
- ✅ 候选人报名时，记录完整推荐链条
