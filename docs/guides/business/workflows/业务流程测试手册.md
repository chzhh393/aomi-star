## 系统简介

**奥米光年**是一个多角色直播主播管理系统，包含以下用户类型：

### 用户角色

1. **候选人** - 应聘主播的人
2. **星探** - 推荐候选人赚佣金的人
   - **星探合伙人（SP）** - 可以发展下级星探
   - **特约星探（SS）** - 不能发展下级
3. **内部员工**
   - **HR** - 审核候选人、分配经纪人
   - **经纪人** - 面试打分、上传资料
   - **管理员** - 最高权限，管理所有功能

### 核心特性

- 微信小程序 + 管理后台（网页版）
- 星探推荐码机制
- 数据脱敏保护
- 多角色权限控制

## 业务流程测试

## 流程1：候选人报名与审核

### 业务场景

候选人通过小程序报名成为主播，经过HR审核、经纪人面试后签约。

### 流程图

```
候选人报名 → HR初审 → 分配经纪人 → 面试打分 → 上传资料 → 最终审核 → 签约
```

### 测试步骤

#### 第1步：候选人报名（小程序）

**操作人员**：候选人微信号

**操作步骤**：
1. 打开小程序
2. 选择"主播报名"
3. 填写个人信息：
   - 基本信息：姓名、性别、年龄、身高、体重
   - 艺名（选填）
   - 联系方式：手机号、微信号
   - 所在地：省、市、详细地址
   - 才艺信息：擅长才艺（可多选）、才艺描述
   - 直播经验：是否有经验、平台、粉丝数、直播名
   - 社交账号：抖音/小红书/B站（选填）
   - 上传照片：至少3张（正面、侧面、全身）
   - 个人爱好：选择标签
4. 点击"提交报名"

**预期结果**：
- ✅ 提示"报名成功，请等待审核"
- ✅ 状态变为"待审核"

**测试用例**：

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| TC-1.1 | 正常填写所有信息并提交 | 提交成功 |
| TC-1.2 | 不填手机号提交 | 提示"请填写手机号" |
| TC-1.3 | 只上传1张照片提交 | 提示"至少上传3张照片" |
| TC-1.4 | 手机号格式错误 | 提示"手机号格式不正确" |

---

#### 第2步：HR初审（管理后台）

**操作人员**：HR账号

**操作步骤**：
1. 登录管理后台
2. 进入"候选人管理"
3. 查看待审核列表
4. 点击某个候选人查看详情
5. 审核内容：
   - 基本信息是否完整
   - 照片质量是否合格
   - 才艺描述是否合理
6. 选择操作：
   - **通过**：点击"通过"按钮
   - **拒绝**：点击"拒绝"，填写拒绝原因

**预期结果**：
- ✅ 通过：状态变为"已通过"，可以分配经纪人
- ✅ 拒绝：状态变为"已拒绝"，显示拒绝原因

**测试用例**：

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| TC-1.5 | 审核通过候选人 | 状态变为"已通过" |
| TC-1.6 | 拒绝候选人并填写原因 | 状态变为"已拒绝"，显示原因 |
| TC-1.7 | 批量选择3个候选人，点击"批量通过" | 3个候选人状态都变为"已通过" |

---

#### 第3步：分配经纪人（管理后台）

**操作人员**：HR或管理员

**操作步骤**：

**方式1：单个分配**
1. 在候选人详情页
2. 点击"分配经纪人"
3. 选择经纪人（显示已分配人数）
4. 点击"确认分配"

**方式2：批量分配**
1. 候选人列表勾选多个候选人
2. 点击"批量分配"按钮
3. 选择目标经纪人
4. 点击"确认分配"

**预期结果**：
- ✅ 分配成功提示
- ✅ 候选人详情显示经纪人姓名
- ✅ 经纪人小程序能看到新分配的候选人
- ✅ 如果候选人之前已分配给其他经纪人，显示"重新分配"提示

**测试用例**：

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| TC-1.8 | 单个分配候选人给经纪人A | 分配成功，经纪人A小程序可见 |
| TC-1.9 | 批量分配3个候选人给经纪人B | 3个都分配成功 |
| TC-1.10 | 将已分配给经纪人A的候选人重新分配给经纪人B | 显示"重新分配"提示，分配成功 |

---

#### 第4步：面试打分（小程序 - 经纪人）

**操作人员**：经纪人账号

**操作步骤**：
1. 打开小程序
2. 进入"候选人列表"
3. 点击某个候选人查看详情
4. 检查看不到的信息（数据脱敏）：
   - ❌ 手机号（已隐藏）
   - ❌ 微信号（已隐藏）
   - ❌ 抖音/小红书/B站账号（已隐藏）
   - ❌ 推荐人信息（已隐藏）
5. 点击"面试打分"按钮
6. 填写评分：
   - **评分结果**（单选）：S级通过/A级通过/B级通过/不通过/待定
   - **评价标签**（多选，5个维度）：
     - 外形：高挑、精致、有气质、清新、性感
     - 才艺：唱歌好、舞蹈好、乐器、表演力强
     - 性格：外向、自信、亲和力、幽默、稳重
     - 表达：口齿清晰、逻辑清晰、应变能力强
     - 潜力：可塑性强、学习能力强、镜头感好
   - **备注**：自由文本
7. 点击"提交评分"

**预期结果**：
- ✅ 提交成功提示
- ✅ 详情页显示评分结果
- ✅ 管理后台可以看到评分信息

**测试用例**：

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| TC-1.11 | 查看候选人详情 | 看不到手机号、微信号、社交账号 |
| TC-1.12 | 给候选人打S级通过，选择5个标签 | 提交成功，详情显示 |
| TC-1.13 | 尝试给未分配的候选人打分 | 提示"无权限" |

---

#### 第5步：上传面试资料（小程序 - 经纪人）

**操作人员**：经纪人账号

**操作步骤**：
1. 在候选人详情页
2. 点击"上传资料"

**上传照片**：
- 点击"上传照片"
- 拍照或从相册选择
- 最多9张
- 点击"提交"

**上传视频**：
- 点击"上传视频"
- 录制或从相册选择
- 最多5个，每个不超过100MB
- 点击"提交"

**预期结果**：
- ✅ 上传成功提示
- ✅ 资料列表显示已上传的照片和视频
- ✅ 管理后台可以查看

**测试用例**：

| 用例 | 操作 | 预期结果 |

| TC-1.15 | 上传10张照片 | 提示"最多上传9张" |
| TC-1.16 | 上传1个视频（50MB） | 上传成功 |
| TC-1.17 | 上传1个视频（150MB） | 提示"视频大小不能超过100MB" |

---

#### 第6步：最终审核（管理后台）

**操作步骤**：

## 流程2：星探推荐

### 业务场景

星探通过推荐码推荐候选人，候选人成功签约后，星探获得佣金。

### 星探层级说明

```
星探合伙人（SP）
├─ 可以发展下级星探
├─ 有推荐码（格式：SC-EXT-20260310-A3B9）
└─ 推荐候选人赚佣金

特约星探（SS）
├─ 不能发展下级
├─ 没有推荐码
└─ 推荐候选人赚佣金
```

### 流程图

```
星探注册 → 获取推荐码 → 分享推荐码 → 候选人扫码 → 候选人报名 → 候选人签约 → 星探获得佣金
```

### 测试步骤

#### 第1步：星探注册（小程序）

**方式1：成为星探合伙人（SP）**

**操作步骤**：
1. 打开小程序
2. 选择"成为星探"
3. 填写个人信息：
   - 姓名
   - 手机号
   - 身份证号
4. 点击"提交注册"

**预期结果**：
- ✅ 注册成功
- ✅ 系统自动生成推荐码（格式：SC-EXT-YYYYMMDD-XXXX）
- ✅ 推荐码显示在个人中心

**方式2：成为特约星探（SS）**

**操作步骤**：
1. 通过SP的推荐码扫码进入小程序
2. 选择"成为星探"
3. 填写个人信息
4. 点击"提交注册"

**预期结果**：
- ✅ 注册成功
- ✅ 自动绑定上级（SP）
- ✅ 没有推荐码（不能发展下级）

**测试用例**：

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| TC-2.1 | 直接注册成为星探 | 注册成功，获得推荐码，成为SP |
| TC-2.2 | 通过SP推荐码注册 | 注册成功，无推荐码，成为SS，显示上级姓名 |
| TC-2.3 | 注册时手机号重复 | 提示"手机号已注册" |

## 流程8：试镜视频上传

### 业务场景

经纪人为候选人上传试镜视频（才艺展示、自我介绍等），存储到云端，管理后台可查看。

### 流程图

```
经纪人进入候选人详情 → 点击上传试镜视频 → 录制或选择视频 → 检查大小 → 上传到云端 → 提交保存
```

### 测试步骤

#### 第1步：上传试镜视频（小程序）

**操作人员**：经纪人账号

**操作步骤**：
1. 经纪人进入候选人详情
2. 点击"上传试镜视频"按钮
3. 进入上传页面
4. 点击"添加视频"：
   - 选择"录制视频"或"从相册选择"
   - 录制视频（建议30秒-2分钟）
   - 或从相册选择视频
5. 查看视频预览
6. 可以删除视频重新选择
7. 点击"提交上传"

**限制规则**：
- 最多上传 3 个视频
- 每个视频不超过 100MB
- 建议时长：30秒 - 2分钟

- ✅ 显示上传进度条
- ✅ 上传成功提示

## 常见问题排查

### 登录问题

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 提示"账号不存在" | 用户名输入错误 | 检查用户名拼写 |
| 提示"密码错误" | 密码输入错误 | 检查密码，注意大小写 |
| 提示"账号待审核" | 申请还未审核 | 联系管理员审核 |
| 提示"账号已拒绝" | 申请被拒绝 | 联系管理员了解原因 |

### 权限问题

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 经纪人看不到候选人 | 候选人未分配 | 联系HR分配候选人 |
| 提示"无权限查看" | 查看未分配的候选人 | 只能查看分配给自己的 |
| 菜单功能灰色 | 角色权限不足 | 联系管理员确认角色权限 |

### 数据问题

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 推荐关系未绑定 | 未通过推荐码进入 | 候选人重新扫码报名 |
| 佣金未显示 | 候选人还未签约 | 等待候选人签约 |
| 统计数据不对 | 数据未刷新 | 退出重新登录 |

### 上传问题

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 视频上传失败 | 文件过大 | 压缩视频或选择较小的文件 |
| 上传进度卡住 | 网络不稳定 | 检查网络，重新上传 |
| 提示"超过数量限制" | 已达到上限 | 删除旧视频后再上传 |
