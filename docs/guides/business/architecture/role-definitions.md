## 1️⃣ 主播候选人（Candidate）

### 定义

已提交报名但尚未签约的应聘者（**也是系统用户**）

### 重要说明

- ✅ **候选人报名时就创建微信账号**（role=candidate）
- ✅ 拥有独立的候选人工作台
- ✅ 可以查看审核进度、接收面试通知
- ✅ 签约后角色自动升级为主播（candidate → streamer）

### 主要功能

- 填写和提交报名表
- 查看审核状态（实时）
- 接收面试通知（小程序消息推送）
- 参加在线面试
- 查看面试结果
- 查看录用通知
- 联系HR

### 状态流转

```
扫码/搜索小程序
    ↓
微信登录 + 填写报名表
    ↓
创建账号（role=candidate）
    ↓
状态：pending → reviewing → interviewed → offered → signed
    ↓
签约后：role升级为streamer
```

### 数据模型

```javascript
{
  _id: 'user_001',
  openId: 'wx_xxx',
  userType: 'candidate',
  role: 'candidate',

  candidateInfo: {
    status: 'pending', // 待审核
    source: 'scout_referral', // 来源：scout_referral / self_apply
    applyData: { /* 报名表数据 */ },
    applyAt: new Date()
  },

  accountStatus: {
    isActivated: true,
    isFirstLogin: false,
    isProfileComplete: true
  },

  profile: {
    name: '张三',
    phone: '13800138000'
  },

  createdAt: new Date()
}
```

### 状态详细说明

| 状态 | 说明 | 候选人可见内容 | 可操作功能 |
|------|------|---------------|----------|
| **pending** | 简历待审核 | "您的简历已提交，HR正在审核中" | 查看简历、修改简历 |
| **reviewing** | 初审中 | "初审进行中，预计3个工作日反馈" | 查看简历 |
| **interview_scheduled** | 已安排面试 | 面试时间、地点、面试官信息 | 查看面试须知、加入面试、改约 |
| **interviewed** | 面试已完成 | "面试已完成，等待最终结果" | 查看面试记录 |
| **offered** | 已发放Offer | Offer详情、薪资待遇、签约须知 | 确认接受、申请协商 |
| **contract_pending** | 待签署合同 | 电子合同内容 | 签署合同 |
| **signed** | 已签约 | "恭喜！即将成为正式主播" | 查看合同、等待入职 |
| **rejected** | 未通过 | 不通过原因（可选）、鼓励语 | 查看反馈、6个月后重新申请 |

## 2️⃣ 主播（Streamer）

### 定义

已签约入职的正式主播

### 主要功能

- 查看本周排班
- 查看培训任务
- 提交练习视频
- 查看直播数据
- 查看收益明细
- 查看周报反馈

### 权限范围

只能查看自己的数据

### 数据模型

```javascript
{
  _id: 'user_001',
  openId: 'wx_xxx',
  userType: 'streamer',
  role: 'streamer',

  // 保留候选人历史
  candidateHistory: {
    appliedAt: '2025-01-05',
    source: 'scout_referral',
    scoutCode: 'SC-EXT-20250102-A3B9',
    upgradedAt: '2025-01-20'
  },

  // 主播信息
  streamerInfo: {
    status: 'active', // active / onboarding / suspended
    stageLevel: 'B', // 级别：S / A / B / C / trainee
    agentId: 'agent_001', // 分配的经纪人
    contractId: 'contract_001',
    joinedAt: new Date(),

    // 统计数据
    totalLiveHours: 120,
    totalIncome: 45000,
    followerCount: 8500
  },

  profile: {
    name: '张三',
    stageName: '小星星', // 艺名
    phone: '13800138000'
  },

  createdAt: new Date()
}
```

### 工作台功能模块

| 模块 | 功能 | 数据来源 |
|-----|------|---------|
| 首页概览 | 本周数据、排班预览 | streamers, schedules |
| 排班管理 | 查看周排班、申请调班 | schedules |
| 培训任务 | 课程列表、练习提交、考核记录 | training_plans, assessments |
| 直播数据 | 观看数据、互动数据、收益数据 | live_sessions |
| 收益明细 | 月收益、历史收益、提现记录 | settlements |
| 个人中心 | 个人资料、合同查看、设置 | users, contracts |

## 3️⃣ 内部员工（Employee）

### 9种角色类型

| 角色 | 代码 | 主要工作 |
|------|------|---------|
| HR管理员 | `hr_admin` | 候选人管理、面试安排、合同签署 |
| 经纪人 | `agent` | 团队管理、排班确认、周报复盘 |
| 运营专员 | `operator` | 排班管理、实时监控、数据分析 |
| 舞蹈培训师 | `dance_teacher` | 舞蹈训练、进度跟踪、考核评估 |
| 化妆师 | `makeup_artist` | 妆容设计、形象指导、效果分析 |
| 搭配师 | `stylist` | 造型设计、服装搭配、风格塑造 |
| 专职星探 | `scout_internal` | 主播招募、候选人筛选 |
| 财务专员 | `finance` | 收益结算、财务审核、发放管理 |
| 摄像师 | `videographer` | 面试录制、素材上传、视频管理 |

### 权限特点

- 🔒 数据访问受限（只能看权限范围内的数据）
- 🎯 功能模块定制（每个角色看到的页面不同）
- 👥 需要邀请码才能注册（防止随意注册）

### 数据模型

```javascript
{
  _id: 'user_002',
  openId: 'wx_yyy',
  userType: 'employee',
  role: 'agent', // 具体角色

  employeeInfo: {
    employeeId: 'EMP-2025-001',
    realName: '李经理',
    department: '经纪部',
    position: '经纪人',
    hireDate: '2024-06-01',
    status: 'active'
  },

  // 经纪人特有数据
  agentInfo: {
    teamSize: 12,
    teamLevel: 'senior', // junior / mid / senior
    managedStreamers: ['streamer_001', 'streamer_002', ...]
  },

  profile: {
    name: '李经理',
    phone: '13900139000'
  },

  createdAt: new Date()
}
```

### 3.1 HR管理员（hr_admin）

#### 主要职责

- ✅ 候选人管理（审核简历、筛选候选人）
- ✅ 面试安排（创建面试、分配面试官）
- ✅ 合同管理（生成合同、管理签署流程）
- ✅ 员工邀请码管理（生成、作废、查看使用记录）

#### 权限范围

- 查看所有候选人信息
- 查看所有面试记录
- 管理所有合同
- 生成员工邀请码

#### 工作台功能

```
- 候选人列表（全部候选人）
- 面试管理（创建、安排、评审）
- 合同管理（模板、生成、签署）
- 邀请码管理（生成、列表）
```

---

### 3.2 经纪人（agent）

#### 主要职责

- ✅ 团队管理（管理12-15名主播）
- ✅ 排班确认（确认运营安排的排班）
- ✅ 培训管理（制定培训计划、考核评估）
- ✅ 周报复盘（填写周报、组织复盘会议）
- ✅ 收益确认（确认主播收益数据）

#### 权限范围

- 只能查看名下主播的数据
- 可以修改名下主播的培训计划
- 可以填写周报和复盘记录

#### 工作台功能

```
- 团队管理（主播列表、详情、目标）
- 排班管理（查看、确认）
- 培训管理（计划、评估）
- 周报复盘（创建、列表、会议）
- 收益确认
```

---

### 3.3 运营专员（operator）

#### 主要职责

- ✅ 排班管理（创建周排班、管理排班）
- ✅ 实时监控（监控直播状态、处理预警）
- ✅ 数据分析（日报、周报、趋势分析）

#### 权限范围

- 查看所有主播的排班数据
- 查看所有直播数据
- 创建和修改排班

#### 工作台功能

```
- 排班系统（创建、管理）
- 实时监控（监控大屏、预警处理）
- 数据分析（日报、周报）
```

## 4️⃣ 外部星探（External Scout）

### 定义

与公司签约的外部合作推荐人

### 主要功能

- 生成专属推荐码
- 分享推荐链接/二维码
- 查看推荐记录
- 查看佣金明细
- 提现申请

### 佣金机制

**签约奖金**：
- 推荐主播入职 → 获得签约奖金（¥500）

**持续佣金**（前3个月）：
- 主播月收益 × 5% = 星探月佣金

**示例**：
```
第1个月：主播收益 ¥10,000 → 星探佣金 ¥500
第2个月：主播收益 ¥15,000 → 星探佣金 ¥750
第3个月：主播收益 ¥20,000 → 星探佣金 ¥1,000
─────────────────────────────────────────
签约奖金：¥500
持续佣金：¥2,250
总计佣金：¥2,750
```

### 数据模型

```javascript
{
  _id: 'user_003',
  openId: 'wx_zzz',
  userType: 'scout',
  role: 'scout_external',

  scoutInfo: {
    scoutCode: 'SC-EXT-20250102-A3B9', // 专属推荐码
    level: 'gold', // bronze / silver / gold / platinum
    status: 'active',
    registeredAt: new Date(),

    // 统计数据
    totalReferred: 15,
    totalSigned: 5,
    totalCommission: 12500,
    paidCommission: 10000
  },

  profile: {
    name: '王推荐',
    phone: '13700137000'
  },

  bankInfo: {
    accountName: '王推荐',
    bankName: '中国银行',
    accountNumber: '6222************1234'
  },

  createdAt: new Date()
}
```

### 工作台功能

```
- 首页（本月数据、推荐码）
- 推荐记录（候选人列表、详情）
- 佣金管理（明细、提现）
- 个人中心（资料、银行卡）
```

## 角色对比表

| 特性 | 候选人 | 主播 | 内部员工 | 外部星探 |
|-----|-------|------|---------|---------|
| **注册方式** | 推荐码/直接搜索 | 从候选人升级 | 邀请码 | 注册申请 |
| **工作台** | 申请进度追踪 | 排班/培训/数据 | 角色专属功能 | 推荐/佣金管理 |
| **数据权限** | 只看自己 | 只看自己 | 受角色限制 | 只看推荐的候选人 |
| **主要目标** | 通过面试、签约 | 完成直播任务 | 完成工作职责 | 推荐主播获取佣金 |
