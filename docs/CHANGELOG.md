## 版本类型说明

- **新增 (Added)**: 新增的功能
- **修改 (Changed)**: 对现有功能的变更
- **弃用 (Deprecated)**: 即将移除的功能
- **移除 (Removed)**: 已经移除的功能
- **修复 (Fixed)**: 任何 bug 的修复
- **安全 (Security)**: 修复安全问题
