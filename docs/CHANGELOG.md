### 新增 ✨

- **经纪人小程序端完整功能** (2026-03-09)
  - 新增经纪人认证工具（用户名密码登录）
  - 新增经纪人登录页面（美观的渐变设计）
  - 重构经纪人主页（显示统计数据和快捷入口）
  - 新增候选人列表页（含筛选：全部/待打分/已打分）
  - 新增候选人详情页（数据脱敏：不显示手机号、社交账号、推荐信息）
  - 新增面试打分页（5档评分 + 5维度标签多选 + 备注）
    - 评分结果：S级/A级/B级/不通过/待定
    - 评价维度：外形、才艺、性格、表达、潜力
  - 新增上传面试资料页（照片最多9张 + 视频最多5个，带上传进度）
  - 新增我的评价记录页（历史评价列表，按时间倒序）

- **候选人信息分享功能** (2026-03-09)
  - 候选人详情页添加微信分享按钮
  - 分享内容自动脱敏处理（仅包含姓名、年龄、身高、城市等基本信息）
  - 不包含敏感信息（手机号、微信号、社交账号、推荐信息等）
  - 分享图片使用候选人照片作为封面

## 版本类型说明

- **新增 (Added)**: 新增的功能
- **修改 (Changed)**: 对现有功能的变更
- **弃用 (Deprecated)**: 即将移除的功能
- **移除 (Removed)**: 已经移除的功能
- **修复 (Fixed)**: 任何 bug 的修复
- **安全 (Security)**: 修复安全问题
