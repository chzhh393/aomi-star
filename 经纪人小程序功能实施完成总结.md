## 🔑 关键技术要点

### 1. 认证系统
```javascript
// 用户名密码登录（区别于微信登录）
await agentLogin(username, password)

// Token自动过期处理
if (tokenExpired) {
  handleTokenExpired()  // 自动跳转登录页
}
```

### 2. 权限控制
- 云函数已实现 ROLE_PERMISSIONS 配置
- 经纪人角色自动脱敏敏感信息
- 只能查看分配给自己的候选人

### 3. 数据脱敏
**经纪人看不到：**
- 手机号（phone）
- 微信号（wechat）
- 推荐信息（referral）

### 4. 文件上传优化
```javascript
// 先显示临时图片，后台上传到云存储
photos = [...this.data.photos, ...tempPaths]
uploadPhotosToCloud(tempPaths)  // 异步上传

// 带进度显示
onProgress: (current, total) => {
  percent = (current / total) * 100
}
```

## 📝 已知限制和建议

2. **视频限制：** 最多5个（可在代码中调整）
3. **视频时长：** 最长60秒（微信限制）

### 未来优化建议

2. **添加缓存机制** - 候选人列表缓存5分钟以减少云函数调用
3. **添加离线支持** - 打分数据临时保存，网络恢复后提交
4. **添加工作量统计** - 管理后台显示每个经纪人的打分数量和效率

## 📞 支持和维护

### 常见问题排查
1. **登录失败** - 检查用户名密码是否正确，检查云函数是否部署
2. **看不到候选人** - 检查是否已分配候选人给该经纪人
3. **上传失败** - 检查云存储权限，检查网络连接
4. **Token过期** - 会自动跳转登录页，重新登录即可

### 技术支持
- 云函数日志：云开发控制台 → 云函数 → 日志
- 小程序日志：小程序开发工具 → 调试器 → Console
- 数据库查询：云开发控制台 → 数据库
